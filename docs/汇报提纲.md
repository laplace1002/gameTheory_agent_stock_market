# 课堂汇报提纲

## 1. 研究问题

我们不再研究普通舆情，而是研究 AI agent 在虚拟股票市场中的交易行为。

核心问题：

1. 不同投资思想的 agent，谁的纸面交易成绩更好？
2. 多个 agent 分开行动好，还是组成团队以后更好？
3. 动态组织机制能不能让表现好的 agent 获得更大话语权？
4. 这种 AI agent 小社会会不会出现可观察的行为规律？

## 2. 项目方法

- 使用历史股票价格数据。
- 给每个 agent 一笔虚拟资金。
- 每个 agent 根据自己的策略做纸面交易。
- 记录收益、波动、最大回撤、夏普比率、交易次数。
- 最后用 dashboard 展示结果。

## 3. 展示重点

展示顺序建议：

1. 展示项目平台主页。
2. 展示不同 agent 的资金曲线。
3. 展示收益率、夏普比率、最大回撤对比表。
4. 解释为什么团队 agent 可能比单个 agent 稳定。
5. 说明项目不是预测股价，而是研究 AI agent 行为规律。

## 4. 一句话结论模板

本项目发现，在纸面交易实验中，不同 agent 的表现差异明显；单一策略往往在某些行情中有效，但在市场风格变化时容易失效。相比之下，团队型 agent 能够综合不同策略意见，在风险控制和稳定性方面通常更有解释价值。
